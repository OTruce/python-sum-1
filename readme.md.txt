News Article Text Analysis

Description

This Python program analyzes a news article stored in a text file ("news-article.txt"). It performs several Natural Language Processing (NLP) tasks, including counting the occurrences of a specific word, identifying the most common word, calculating the average word length, counting the number of paragraphs, and counting the number of sentences.

Features

Counts the occurrences of a specified word.
Identifies the most common word in the article.
Calculates the average word length.
Counts the number of paragraphs.
Counts the number of sentences.
Handles edge cases such as empty text input.
Requirements

Python 3.x
A text file named "news-article.txt" containing the news article in the same directory as the Python script.
How to Run:

Place "news-article.txt" in the project folder.
Open the project in Visual Studio Code or another Python IDE.
Run the Python script.
Follow the on-screen menu to select the desired text analysis option.
